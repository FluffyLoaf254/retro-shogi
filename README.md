# Retro Shogi

I've archived this project, but it can still be used as an example of how to do 3D screen picking in WebGL2.

A shogi game with retro graphics rendered with WebGL2.

## How to Run
Git clone the repo and then run the following NPM commands to run the project:
```
npm run build
npm run serve
```
See the results by going to http://localhost:8080.
